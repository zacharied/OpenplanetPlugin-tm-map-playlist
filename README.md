# Map Playlist

Create a playlist of maps to play in solo. Switch to the next map on demand, when you obtain a specific medal, or after the timer is up.

You can save multiple playlist to use later, reorder the maps, randomize them, among many other features.

## Features

* Add maps from 10+ different sources
* Support for Race, Stunt, Platform, and Royal maps
* Randomize the map order
* Create multiple playlists
* Add / Remove tags to maps and playlists
* Max timer per map
* Goal medal
* Load maps in the editor
* Track PB / Session PB
* Hotkeys
* Support for [Warrior Medals](https://openplanet.dev/plugin/warriormedals)
* and a lot of customization settings!

## Sources

You can use the following sources:

* TMX map ID
* TMX mappack ID
* Map UID
* Favorites
* File files
* Folders
* Club campaign
* Seasonal campaign
* Weekly shorts
* Weekly grands
* TOTD month
* URLs

For file and folder paths, you need to provide an absolute path!

For URLs, the plugin supports the following links:

* TMX map
* TMX mappack
* Trackmania.io map leaderboard
* Trackmania.io campaign (seasonal, weekly shorts / grands, and club)
* Trackmania.com track
* Trackmania.com club campaign
* Trackmania.com seasonal campaign
